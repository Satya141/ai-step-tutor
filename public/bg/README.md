# Background Images

Add subject-specific background images here:

- `maths.jpg` - Mathematics theme (e.g., equations, graphs, geometric patterns)
- `physics.jpg` - Physics theme (e.g., atoms, circuits, motion)
- `chemistry.jpg` - Chemistry theme (e.g., molecules, periodic table, lab equipment)
- `biology.jpg` - Biology theme (e.g., cells, DNA, nature)

## Recommended Image Specifications

- **Resolution**: 1920x1080 or higher
- **Format**: JPG or PNG
- **Style**: Abstract, subtle, not too busy
- **Color**: Match subject theme (blue for maths, purple for physics, green for chemistry, teal for biology)
- **Opacity**: Will be overlaid with semi-transparent white container, so can be vibrant

## Free Image Sources

- [Unsplash](https://unsplash.com)
- [Pexels](https://pexels.com)
- [Pixabay](https://pixabay.com)

## Example Search Terms

- Maths: "mathematics abstract", "geometric patterns", "equations background"
- Physics: "physics abstract", "atom structure", "energy waves"
- Chemistry: "chemistry lab", "molecular structure", "colorful liquids"
- Biology: "biology cells", "DNA structure", "nature microscopic"

**Note**: The app will work fine without images (falls back to gray-50 background). Add them when ready for presentation!
